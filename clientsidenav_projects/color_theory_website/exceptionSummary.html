How I inserted exceptions in the source code of The Math of Colors

In the Math of Colors site, the user is asked for three values between 0-255 in a form. I wanted to put exceptions that would restrict users from entering character values and stop results from printing before the error message for values exceeding the min and max in the form.


Heres what I did...

I put a try statement testing the values of the user input. The form already corrects the user on invalid numbers and characters, but now an error message exists on the page to remind the user what restraints are in use. When the user leaves any input "", the page restarts. Now if the user does not enter a valid number, there will be consequences. This makes the code for this tight.
